Sou desenvolvedor de Sistemas Web, formado em Análise e Desenvolvimento de Sistemas, iniciei na área de informática em 2006 fazendo reparos em computadores desktops.

A cada dia tive uma nova curiosidade e interesse pela área de Tecnologia. Atuei como técnico de informática e posteriormente analista de suporte e consultor de TI.

Em 2010 resolvi atuar na área de desenvolvimento. Iniciei como estagiário, atuei em uma empresa do ramo imobiliário na sustentação de Projetos, como: sistemas de repasses imobiliários, extranet de Vendas, intranet e portais Web.

Fui trabalhar na Administração Pública atuei na sustentação e desenvolvimento de projetos como: Monitora Osasco, Plataforma de transparência Visão 360, Plataforma de Consulta Pública Participa Osasco, participei do projeto de aplicativo do Visão 360 fornecendo uma API com os Dados da plataforma e hotsite Acolher Osasco.

SERVIÇOS DISPONÍVEIS

E-commerce

Desenvolvimento de Loja Virtual usando Bootstrap 4, Linguagem C#, tecnologias Asp.Net MVC, Entity Framework e Dapper, SQL Server, JavaScript, HTML5 e CSS3. Criamos um Layout responsivo, fazemos as integrações de operadoras de pagamento como o PagSeguro.

Sistemas Web

Desenvolvimento de Sistemas Web, Portais Web, Intranets e HotSites. Usando tecnologias Asp.Net WebForms ou MVC, Banco de Dados SQL Server ou My SQL, usamos Frameworks como Bootstrap e MaterializeCSS. Construímos um Layout Responsivo e voltado para o segmento do Cliente.

Aplicativos

Desenvolvimento de aplicativos iOS e Android, com base nas necessidades específicas do Cliente. Usamos plataformas nativas para desenvolvimento, como XCode e Android Studio.
